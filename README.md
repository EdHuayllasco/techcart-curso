# TechCart. Curso FrontEnd y React

Repositorio del alumno. Construimos una tienda en línea, TechCart, clase por clase. Cada
carpeta `clase-XX/` contiene el código de esa clase, sus diapositivas en PDF y los
ejercicios para practicar.

## Cómo usar este repositorio

1. Entra a la carpeta de la clase, por ejemplo `clase-01/`.
2. Lee su `README.md`: qué vimos y cómo ver la página.
3. Abre el código y resuelve los ejercicios (están dentro del propio código).
4. Sube tus avances con `git add .`, `git commit` y `git push`.

## Clases

- Clase 01. La web y el HTML: semántica y formularios. Disponible.
- Clase 02. Introducción a CSS: selectores, especificidad y modelo de cajas. Próxima.

## Herramientas

- Visual Studio Code (code.visualstudio.com).
- Un navegador: Chrome o Edge.
- Git y una cuenta de GitHub para subir tu trabajo.
- Extensión recomendada: Live Server.

Node.js y los frameworks se usan más adelante (desde la Semana 3 y en React). En las
primeras clases basta con abrir los archivos `.html` en el navegador.

Los datos e imágenes de demostración provienen de DummyJSON (dummyjson.com).

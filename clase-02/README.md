# Clase 2. Introducción a CSS

Esta es la próxima clase. El código y las diapositivas se publicarán aquí después de la
sesión. Por ahora, estos son los temas que veremos.

En la Clase 1 dejamos la tienda sin diseño. En la Clase 2 le damos sus primeros estilos y
aprendemos las bases de CSS.

## Temas a tocar

1. Qué es CSS y cómo se aplica: estilos en línea, `style` interno y hoja externa con
   `link` (la forma recomendada). Anatomía de una regla: `selector { propiedad: valor; }`.
2. Selectores: de etiqueta, de clase, de id, descendientes y de estado (`:hover`).
3. Especificidad: por qué una regla le gana a otra (id sobre clase, clase sobre etiqueta).
4. Modelo de cajas: `content`, `padding`, `border` y `margin`; `box-sizing`; centrar con
   `margin: 0 auto`.

## Qué construiremos

Daremos estilo al `index.html` de la Clase 1: colores, tipografía y las tarjetas de
producto, que empezarán a verse como una tienda de verdad.
